# DSA Games Hub

A Java-based game collection demonstrating Data Structures and Algorithms concepts.

## Features
- Sudoku (Backtracking Algorithm)
- Memory Game (Stack/HashMap)

## Requirements
- Java 17+
- Maven
- javaFX

## How to Run
1. Clone this repository
2. Run 
3. `mvn clean`
4. `mvn compile`
5. `mvn javafx:run`
